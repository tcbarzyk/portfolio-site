# Portfolio Site

Personal portfolio built with Astro, featuring a canvas-based topographic hero and dynamic content sections.

## Tech Stack

- Astro
- TypeScript
- Canvas API

## Project Structure

- `src/components/TopographicHeroRefactored.astro` — main hero component
- `src/styles/hero.css` — hero and project styling
- `src/data/` — content data (about, projects, contact, resume)
- `public/projects/` — project images

## Getting Started

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Preview

```sh
npm run preview
```
